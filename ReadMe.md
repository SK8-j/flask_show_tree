🎁 **项目简介**  
使用flask框架去展示一个礼盒动效，点开是圣诞节的祝福视频

🎄 **项目结构**  
- `app.py`：主程序文件
- `templates/`：HTML模板文件夹
- `static/`：静态资源文件夹

📦 **安装指南**  
1. 克隆仓库：`git clone https://github.com/SK8-j/flask_show_tree.git`
2. 安装依赖：`pip install flask`
3. 运行项目：`python app.py`

🎉 **使用说明**  
打开浏览器，访问`http://127.0.0.1:5000`，点击礼盒查看圣诞节祝福视频
